## 숫자의 합

##### 문제설명
> N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.<br>
첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.<br>
입력으로 주어진 숫자 N개의 합을 출력한다.<br>


##### 입출력 예
|input|answer|
|:---:|:---:|
|1<br>1|2|
|5<br>54321|15|
|11<br>10987654321|46|
